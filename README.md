🛒 Online Grocery Mart - MERN Stack Project
An end-to-end web application for a modern online grocery shopping experience, built with the MERN stack (MongoDB, Express.js, React.js, Node.js). This platform allows users to browse groceries, add them to the cart, and securely place orders — all with a responsive and intuitive interface.

🚀 Features
🧑 User Authentication (JWT-based login/register)

🛍️ Browse products by category

🛒 Add to cart / Remove from cart

📦 Place orders and view order history

🔐 Role-based access (Admin & User)

⚙️ Admin Dashboard:

Manage products (Add/Edit/Delete)

Manage categories

View orders

📱 Fully responsive (mobile-first design)

🌐 RESTful API with secure backend
